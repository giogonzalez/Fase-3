Introduccion HTML


Son las siglas en inglés de (HyperText Markup Lenguage) que significa Lenguaje de Marcado de HiperTexto. este lenguaje de programación es empleado para realizar páginas Web y funciona como estándar de referencia para la codificación y estructuración de las mismas.

El W3C (World Wide Web Consortium) es una organización encargada de la estandarización de los parámetros de Red. En donde se emplea el html como el más importante lenguaje web y que prácticamente todos los navegadores lo han adaptado.

Este código cuenta con la caracteristica de ser liviano y meramente textual, pero permiter insertar direcciones URL de las imágenes, audios, videos que serán observados por el navegador para ensamblar la página, así como las indicaciones para la representación gráfica y estética del texto que se encuentre en la misma.

HTML cuenta con diversas versiones y modificaciones, avanzando hacia un modelo de funcionamiento más eficiente y veloz, que obliga a las distintas empresas de software y navegadores a aplicar consecutivos parches a sus productos para ponerlos al día con cada nueva versión.

Introduccion CSS


CSS (hojas de estilo en cascada) es un lenguaje básico y esencial para crear páginas web , el se encarga de la parte de la presentación, mostrar los elementos de la página, posición, forma, espacio, colores en conclusion toda la parte estética. CSS es un lenguaje que consiste en una serie de elementos mediante los cuales se declaran los estilos.

El W3C define las especificaciones del estándar CSS. para los navegadores segun las especificaciones, para que las páginas web se vean igual en todos ellos.

Los navegadores cada vez mas cumplen con el estándar CSS Para esto se recomienda que se debe maquetar las páginas web siguiendo el estándar CSS y no las especificaciones de un navegador determinado. Ademas es importante mantener actualizado el navegador para las funcionalidades que antes no tenían.
